A fast-paced maze runner game built with vanilla JavaScript + Canvas. No dependencies.

Open `index.html` in any browser, or play on GitHub Pages.

## Features

- Procedurally generated mazes with increasing difficulty
- Red death line chasing you from above
- Collect coins for score multipliers
- 3 Power-ups: Shield, Time Slow, Magnet
- Combo system (up to 12x!)
- 5 stages with unique themes
- 6 achievements to unlock
- Weekly leaderboard (localStorage)
- Mobile touch controls

## Controls

- **Desktop**: Arrow keys / WASD
- **Mobile**: Swipe or on-screen buttons
